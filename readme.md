# 联合同盟宣言序言
![](https://img.shields.io/badge/translated-0%25-orange)  
## 其他语言
* [English](./readme_en.md)  

在我上小学的时候，我的梦想是成为一名科学家造福人类，让人们都过上美好生活  
但到我初中时，感觉当科学家并不能实现此目的  
2016年，我领悟出一个理论——联合同盟理论  
本文将阐述这一理论  

简言之，人类若不团结统一终将毁于内战  

## [正文](./ULT_Manifesto.md)  

master分支很久才会更新一次，最新未确定内容请看[develop](https://github.com/Tenportion/ULT_Manifesto/tree/develop)分支  

本文分为三个部分，团结、维稳、优化  
**团结**是必要的，不然一定会毁于内战  
**维稳**同样是必要的，不然团结的成果一样会被破坏，不过优先级稍低  
**优化**是人类迈向美好生活的道路，属于锦上添花  
**Optimization** is the road for mankind to a better life, which is the icing on the cake.  
所以本文的**团结**部分为最核心观点；**维稳**部分是重要目标，方法仅供参考；**优化**部分更多只是建议  

因此只要大致看过并认可本文的**团结**部分，就可以骄傲地自认为是联合同盟的一员  
联合同盟不是一个国家，也不是一个党派，而是一个松散而自由的思想团体
